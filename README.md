## Udacity前端项目 -- 订阅阅读器测试

这是[Udacity](https://cn.udacity.com)前端开发（进阶）课程的第四个项目，利用Jasmine测试一个已完成的订阅阅读器。

### Getting started

#### Usage

本项目暂未打包，直接在浏览器中运行index.html即可

### Testing overview

这里介绍了项目利用Jasmine测试的内容：

- 测试是否定义了订阅源，且所有订阅源是否都有名称和订阅链接字段
- 测试侧边栏菜单功能是否正常
- 测试获取订阅内容功能是否正常，且切换新源时内容是否改变
